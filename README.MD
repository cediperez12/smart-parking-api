# Smart Parking API

## How to Run
Enter the following command
```
.\mvnw spring-boot:run
```

## How to build
Enter the following command
```
.\mvnw clean install
```


## API Documentation
You may check http://localhost:8080/public/swagger-ui/index.html
after running to check the API documentation.
